### OOP의 특징
> - 캡슐화<br>
>   : 객체의 변수, 메소드 등 실제 구현 내용을 보이지 않게 감싸는 개념<br>
> - 추상화<br>
>   : 사물들의 공통적인 특징을 파악해서 하나의 개념으로 다루는 것<br>
> - 다형성<br>
>   : 같은 자료형에 여러 가지 객체를 대입하여 다양한 결과를 얻어내는 성질<br>
> - 상속성<br>
>   : 클래스로부터 속성과 메서드를 물려받는 것<br>
